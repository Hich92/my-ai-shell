# CONSTITUTION DU PROJET AI-SHELL

- **ARCHITECTURE** : Développement strictement modulaire dans `/src/modules/`.
- **ISOLATION** : Un module ne doit jamais importer un composant d'un autre module (découplage total).
- **DATA** : Utilisation de types TypeScript stricts. Interdiction du type 'any'.
- **UI** : Utilisation exclusive de Tailwind CSS et des composants Shadcn (dossier `/src/components/ui/`).
- **AGENTS** : Se référer systématiquement au fichier `AGENTS.md` pour respecter les standards Next.js 16.
